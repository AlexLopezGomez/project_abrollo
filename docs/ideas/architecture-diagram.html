Monte Carlo Simulation
Simula 1,000 "años posibles del mundo". En cada iteracion tira los dados para cada evento raiz, propaga consecuencias por el grafo, y obtiene el retorno de cada empresa. Resultado: una matriz de 1,000 filas x 100 columnas (escenarios x empresas).
